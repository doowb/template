# template [![NPM version](https://badge.fury.io/js/template.svg)](http://badge.fury.io/js/template)

> template-next, plus another `-next` afterwards.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i template --save
```

## Usage

```js
var nextNext = require('template');
```
